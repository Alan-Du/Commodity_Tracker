# Commodity_Tracker---A python based commodity tracking and research system
This is a commodity automatic tracking and analyzing system based on PYTHON 3. System supports automatic data download and upload from commodity exchanges, St Louis Fed data center and Yahoo Finance data.
# Installation
To install this project, please use PYTHON 3 and MySQL database. The detailed databse setup located in "DB" folder as "dbLogin.py" where you can find all login information to be settled with your own system. Aafter download the whole project, suggests to use a PYTHON based IDE for example "Spyder" to load in. And a directly test case is provided in Test_Sys folder.
# Usage
This system is focused on web-scrapping from commodity exchanges (Zhengzhou, Shanghai, Zhongjin, Dalin) and St Louis Fed as well as Yahoo Finance data center. Store them into local database for future usage. Also generate different comparsion reports for commodity markets and compared with stock market. Also it allows user designed comparion and report.
